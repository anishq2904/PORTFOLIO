📌 Description
A visually appealing and interactive personal portfolio website built using HTML5, CSS3, and JavaScript. This portfolio showcases my projects, skills, certifications, education, and contact information — all within a responsive, modern UI.

📌 Features
📱 Responsive Design — mobile-friendly and desktop-ready

🎨 Animated Backgrounds and Scroll Effects

📝 About Me Section with Career Objective

💻 Technical and Soft Skills Display

🎓 Education Timeline & Certifications

🚀 Project Highlights with description cards

📞 Contact Section with direct links for phone, email, LinkedIn, and GitHub

🎉 Floating hearts and cursor trail animations for a lively experience

📌 Technologies Used
HTML5

CSS3 (Custom gradients, animations, transitions)

JavaScript (ES6) for scroll effects, cursor trails, and dynamic animations

Google Fonts (Montserrat)

Custom icons and emojis for visual appeal

📌 How to View / Run
Download or clone this repository

Open anishq_portfolio.html directly in your web browser
